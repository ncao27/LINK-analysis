# A Project on Analyzing the LINK Dataset 

## 🌐 About LINK
The dataset contains neural data recorded over a period of 3.5 years on a single non-human primate performing dexterous tasks. In total there are 312 sessions with around 350-400 trials per session (trial meaning a period of time where the primate attempts some dexterous movement).
